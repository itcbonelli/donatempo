◆ controllaTempo()

static controllaTempo ( int  $scarto = 900)
static

Controllo di sicurezza Richiede al client di inviare un parametro _ts con il timestamp corrente.

Questo controllo può essere utilizzato per contrastare alcuni attacchi di tipo DoS. Funziona solo se l'orologio di sistema del client e del server sono allineati, con uno scarto temporale minimo.

Parametri
int$scartoscarto temporale ammesso. Default 900 secondi (15 minuti)
